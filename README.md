# Portfolio_site# Personal Portfolio Landing Page

Live URL: Coming soon

## Overview
This is a responsive personal portfolio website built to showcase:
- Introduction
- About
- Skills
- Projects
- Certifications / Achievements
- Contact information

## Features
- Responsive layout (mobile, tablet, desktop)
- Sticky navbar with section links
- Mobile hamburger menu
- Smooth scrolling navigation
- Dark mode toggle
- Scroll reveal animations

## Tech Stack
- React
- Vite
- CSS3
- JavaScript (ES6)

## Project Structure
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`
- `vite.config.js`
